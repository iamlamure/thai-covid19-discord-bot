# Thai Covid-19 Bot

บอทให้ข้อมูลสำหรับรายงานสถานการณ์ Covid-19 ในประเทศไทย
  
พิมพ์ `/covid` เพื่อดูรายงานปัจจุบัน  
พิมพ์ `/covid sub` เพื่อติดตามข่าวอัตโนมัติทุกวันเวลา 19.00 น.  
พิมพ์ `/covid unsub` เพื่อยกเลิกการติดตามข่าว
